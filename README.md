# PowerView

Smart Energy Intelligence Socket System

## Overview
PowerView monitors real-time electricity usage, calculates cost, and enables remote control of appliances.

## Modules
- ESP32 Smart Socket Firmware
- FastAPI Backend
- Supabase Database
- Next.js Dashboard

## Core Features
- Real-time power monitoring
- Energy billing (GH₵)
- Remote ON/OFF control
- Smart alerts
- Device provisioning system
